## Dev

### 1️⃣ Clone the repo
```bash
git clone https://github.com/1andgraf/FL-Messenger.git
cd messenger-app
```

### 2️⃣ Install dependencies
```bash
npm install
# or
yarn install
```

### 3️⃣ Add Firebase config  
Create `firebase.js` file and put your Firebase credentials inside.

### 4️⃣ Run the app
```bash
npm start
```
